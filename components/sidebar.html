Tentang

Shifted. adalah alat transformasi koordinat geodetik untuk konversi antara datum Indonesia (ID74) dan sistem referensi global WGS 84 secara akurat dan real-time langsung pada peta interaktif.

Datum yang Didukung
🌐 WGS 84 🇮🇩 ID 74 🏙 DKI Jakarta
Metode Transformasi

Menggunakan parameter Bursa-Wolf 7-parameter untuk transformasi antara ellipsoid Bessel 1841 (ID74) ke GRS 1980 / WGS 84 dengan akurasi sub-meter.

Parameter ID74 → WGS84
ΔX  : −587.8 m
ΔY  : +519.8 m
ΔZ  : +145.7 m
Src : Bessel 1841
Dst : WGS 84 / GRS80
